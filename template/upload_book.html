How to Upload Book

1. Name the cover image file to match the name of the book appointed by the author.

and must be a JPG file only (other files are not allowed)

2. Upload the image file and press upload file. If the image is uploaded successfully,

a pop-up "success" will appear. It is considered that the image upload is complete.

3. Fill in the book name, book type, price, intro and content.

4. Press submit. If the upload is successful, a pop-up "success" will appear.

The upload is considered completed.
